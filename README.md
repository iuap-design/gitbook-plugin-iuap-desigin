# gitbook-plugin-iuap-desigin

## 说明

gitbook-plugin-iuap-desigin是基于gitbook的扩展功能插件。

## 功能介绍

- 支持左侧目录展开
- 引入 `iuap-desigin` 相关资源时，可同时将生成的 `html` 完美融入 `iuap-desigin` 官网。

## 使用

```
npm install gitbook-plugin-iuap-desigin --save-dev
```

`gitbook` 的 `book.json` 配置示例

```
"language":"zh-hans",
"plugins":[
  "iuap-desigin"
],
```
